# Mistake Log

A minimal, aesthetic web app for tracking LeetCode mistakes during interview prep. Built to help identify patterns in what trips you up, so you can focus your practice where it matters.

## Features

- **Log entries** with problem name, topic, difficulty, status, and notes
- **Track what went wrong** and the key insight that made the solution click
- **Set redo dates** with overdue warnings so nothing slips through
- **Insights dashboard** showing solve rate, difficulty breakdown, and weak spot ranking
- **Filter** by topic or difficulty to review specific areas
- **Export/Import** your data as JSON for backups or switching devices
- **Fully local** — all data stored in your browser, nothing sent anywhere

## Usage

Visit the live site: `https://yourusername.github.io/mistake-log/`

Or run it locally — just open `index.html` in your browser. No install, no build step, no dependencies to manage.

## How it works

It's a single self-contained HTML file with React loaded from a CDN. Your data is saved to `localStorage`, so it persists between sessions but stays on your device. Use the export button to back up your data periodically.

## Topics covered

Arrays, Strings, Hashmaps, Two Pointers, Sliding Window, Binary Search, Linked Lists, Stacks/Queues, Trees, Graphs, BFS, DFS, DP, Greedy, Backtracking, Sorting, Heap, Trie, Union Find
